= View API Instance Contracts 
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]
:keywords: view contracts, api, contracts, viewing

You can view the contracts for your API instance from API Manager after you grant access to applications.

To view your API instance contracts:

. In API Manager, click *API Administration*.
+
image::api-manager-administrator.png[api-manager-administrator]
+
. Under the API Name, click the ">" to expand the API instances list for the API.
. Click the `<api version>:<api instance>` link, for example, "v1:16046365". 
. From the configuration page for the API instance, click *Contracts* from the left menu. 
+
This tab displays the contract information for your API instances.

image::api-manager-client-app-v2.png[contracts]
